# Government Funding Prediction – India

## 📊 Overview
This project predicts state-wise government funding allocations for 2025–26 using machine learning models based on historical data.

## 🎯 Objective
- Analyze funding data
- Build ML models
- Compare performance
- Generate insights

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## 🤖 Models Used
- Random Forest (Best Model)
- Linear Regression
- KNN
- Decision Tree
- K-Means Clustering

## 📈 Results
- Random Forest: MAE = 3.48, Correlation = 0.96
- Linear Regression: MAE = 9.15
- KNN: MAE = 19.74
- Decision Tree: Overfitting observed

## 🔍 Key Insights
- Assam has highest predicted funding
- Clear clustering into 3 groups (Low, Medium, High)
- Random Forest performed best

## ⚠️ Limitations
- Only 3 years of data
- Missing values handled using median

## 🚀 Future Improvements
- Add more data
- Use time-series models
- Include GDP/population features
